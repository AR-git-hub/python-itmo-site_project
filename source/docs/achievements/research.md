# Исследовательская деятельность

## Научные проекты

### Криптография: реализация шифра Хилла и кода Хэмминга
Исследование и программная реализация классических алгоритмов шифрования и помехоустойчивого кодирования.
Протестирована работа шифра Хилла для матриц различной размерности, реализован код Хэмминга (7,4) с возможностью обнаружения и исправления ошибок.

**PDF отчет:** [PDF отчет](../materials/pdf/Math.pdf)


### RAG-системы на базе GigaChain
Разработка и исследование Retrieval-Augmented Generation системы с использованием GigaChain для ответов на вопросы по научным документам. Создание векторного хранилища, интеграция с LLM, оценка качества ответов.

Руководитель: Авдюшина Анна Евгеньевна

**DOCX отчет (скачать)** [Docx](../materials/docx/ЛР 3. Рябков Александр Владиславович.docx)

### Промпт-инжиниринг для генерации эссе
Исследование влияния формулировок промптов, ролей, техник промпт-инженерии на качество текстов, генерируемых большими языковыми моделями.

Руководитель: Авдюшина Анна Евгеньевна

Выявлены оптимальные структуры промптов для эссе.
