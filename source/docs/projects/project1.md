# **Проект 1. Статический сайт на MkDocs с CI/CD**

## **Цель и описание работы**
Создание статического сайта-портфолио с использованием MkDocs и на хостинговой платформе GitHub Pages, дополнительно включающего в себя информацию и отчеты к выполненным лабораторным работам курса Python Дополнительно применялась практика CI/CD посредством GitHub Actions (.yml кода)

## **Реализация и Функционал**
- MkDocs: реализация стандартного стека фронтенда (HTML+CSS) на основе .md файлов, отображающих структуру и содержание страниц
- MkDocs. Material: CSS реализация на основе существующей Material теме. Добавлена возможность темной темы на светлую. Темная тема: Material slate, светлая: Material default (подробная конфигурация изложена в кодовой базе)
- CI/CD: GitHub Actions. Реализация практик непрерывной интеграции и деплоя на основе .yml файла (deploy.yml) и встроенных GitHub инструментов (Actions)
- Публикация сайта: GitHub Pages. Использовалась, как возможность получения бесплатной возможности хостинга без написания бэкенд составляющей сайта.

## **Организация рабочего пространства и Кодовая база**

Структура репозитория:

```
.github/ – Actions
    workflows/
        deploy.yml
source/ – исходные файлы
    .nojekyll – отключение автоматической обработки файлов движком Jekyll
    mkdocs.yml – конфигурационный файл сайта
    docs/ – исходники и материалы страниц
        index.md – главная
        about.md – об авторе
        achievements/ – достижения
            academic.md – академические
            cultural.md – культурно-творческие
            research.md – исследовательские
            social.md – общественно-организаторские
        projects/ – лабораторные работы
        materials/ – материалы, наполняющие текстовые страницы
            docx/ – файлы Word
            img/ – изображения
            pdf/ – pdf-файлы
        
```

### Конфигурация цветовой палитры

Исходная тема: Material default.
```
theme:
  name: material
  palette:
```

### Темная тема

```
- scheme: slate
  primary: deep purple
  accent: pink
  toggle:
    icon: material/weather-night
    name: Переключить на светлую тему
```

### Светлая тема

```
palette:
# Светлая тема (Material default)
- scheme: default
  primary: indigo
  accent: indigo
  toggle:
    icon: material/weather-sunny
    name: Переключить на тёмную тему
```
